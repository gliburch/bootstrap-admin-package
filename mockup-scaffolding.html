Hello! 안녕하세요!

이 페이지는 Bootstrap v3.2.0 기반으로 관리자 화면을 구성하는데 필요한 컴포넌트를 쉽게 활용할 수 있도록 목업(Mock-up)화 한 문서입니다. 기본 Bootstrap CSS, Documents를 참고하여 모든 HTML을 작성하였으며, 필요에 따라 관련 플러그인을 일부 사용하였습니다.

Filter 필터

관리자 화면에선 상상을 초월하는 다양한 검색 필터 양식이 요구됩니다.

Tab 탭

탭은 항상 자주 등장하는 컴포넌트입니다.

Buttons 버튼

다양한 색상과 종류의 버튼을 지원합니다.

Table 테이블

대개 관리자 화면 에서는 보통 한 화면 내에 최대한 많은 정보를 담아야 하는 요구사항이 많은 만큼, 높이가 낮고 행간 구분이 편리한 className 을 기본으로 채택하였습니다.

테이블 컬럼이 우리가 원하는 대로 너비를 적절하게 스스로 잡아주면 좋겠지만 그렇지 않은 경우, 테이블 셀 정렬이 각양각색인 경우 참고할 수 있는 다양한 className 예시입니다.

Table with Header and Footer 옵션 정보/기능이 포함된 테이블

관리자 화면에서 자주 사용되는 테이블 정렬 및 레코드 제어 기능을 구현하기 위한 구성입니다.

Ultra Densed Table 빡빡한 테이블

경우에 따라서는 매우 많은 양의 데이터를 한꺼번에 노출시키는 바람에 화면을 벗어날 수 밖에 없는 테이블도 왕왕 구현하게 됩니다. 모든 내용을 한 화면에 담을 수는 없지만, 횔 스크롤을 통해 테이블의 형태를 보존하고 탐색이 가능하게끔 조치를 취해줘야 할 수도 있습니다.
